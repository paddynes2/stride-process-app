# Golden Paths Suite — Stride

## Purpose
Verify the critical user journeys end-to-end. Run at phase completion.

## Path 1: New User Onboarding (10 actions)
1. [ ] Sign up with new email
2. [ ] Verify redirect to workspace canvas
3. [ ] Getting Started template visible
4. [ ] Add a section to canvas
5. [ ] Add a step inside section
6. [ ] Click step → edit name and status
7. [ ] Add maturity score
8. [ ] View heat map toggle
9. [ ] Navigate to gap analysis
10. [ ] Verify gap data reflects step maturity

## Path 2: Journey Canvas Workflow (8 actions)
11. [ ] Create new journey tab
12. [ ] Verify journey canvas renders
13. [ ] Add a stage
14. [ ] Add a touchpoint in stage
15. [ ] Click touchpoint → edit pain/gain scores
16. [ ] View heat map (pain coloring)
17. [ ] Export journey as PDF
18. [ ] Verify PDF generates

## Path 3: Collaboration Features (7 actions)
19. [ ] Navigate to teams page
20. [ ] Create team + role + person
21. [ ] Assign role to a step
22. [ ] View cost calculations
23. [ ] Create public share
24. [ ] Open public share URL (unauthenticated)
25. [ ] Verify read-only canvas renders

## Path 4: Perspectives Workflow (5 actions)
26. [ ] Navigate to settings → perspectives
27. [ ] Create a perspective (name, color, icon)
28. [ ] Switch active perspective in header
29. [ ] Add annotation to a step
30. [ ] Verify canvas node shows perspective indicator

## Pass Criteria
All 30 checks pass. No console errors during any path.
